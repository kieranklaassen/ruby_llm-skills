# Greeting Generator

Generate warm, personalized greetings for any occasion.

## Usage

When generating greetings:
1. Consider the recipient's name
2. Match the tone to the occasion
3. Keep it concise but warm

## Examples

- "Hello [Name], welcome aboard!"
- "Good morning [Name], hope you have a great day!"
